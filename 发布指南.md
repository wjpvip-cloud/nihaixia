# 倪海夏Skill发布到GitHub - 喂饭级指南

## 第一步：打开终端
按 `Cmd+Space` 输入「终端」打开。

## 第二步：进入技能目录
```bash
cd ~/Desktop/nihaixia-skill
```

## 第三步：登录GitHub（如果还没登录过）
```bash
gh auth login
```
**按提示操作：**
1. 选「GitHub.com」
2. 选「HTTPS」
3. 选「用浏览器登录」（会自动打开浏览器）
4. 浏览器授权后回到终端

## 第四步：运行自动发布脚本
```bash
./deploy.sh
```

## 如果遇到问题

### 1. 提示「gh命令未找到」
安装GitHub CLI：
```bash
brew install gh
```
或去 https://cli.github.com/ 下载安装

### 2. 提示「仓库已存在」
正常，直接推送更新即可。

### 3. 推送时要求输入用户名密码
改用token登录：
```bash
gh auth logout
gh auth login --with-token
```
然后粘贴你的GitHub Personal Access Token（需要有repo权限）

## 验证发布成功
1. 浏览器打开：https://github.com/wjpvip-cloud/nihaixia
2. 看到 skills/nihaixia.md 文件
3. 文件内容正确

## 安装测试
在Claude Code里测试：
```bash
npx skills add wjpvip-cloud/nihaixia-skill
```
然后输入 `/nihaixia` 测试。
